# PureTech——Java后端学习项目

## 项目目的

PureTech是一个Java后端学习项目，旨在记录自己学习后端技术，如果有可能帮助到其他人，也不妨是一件有意义的项目。通过构建一个完整的后端应用来学习和掌握Java后端开发的核心概念和技能。

2026年研究生考试结果已经出了，我拟录取了一所211大学。在家里休息了1个月之后，决定学习Java后端开发相关的技术，争取在暑期找一份相关实习，积累一些实战经验。

为此，我通过学习**二哥**的开源项目——技术派，来进行学习！我这里东施效颦，为项目取名为**PureTech**，纯技术的意思，哈哈哈。

## 思考与总结

为了更好地开始这个项目，我们需要先知道这个项目要解决什么问题，提供什么价值才行！
